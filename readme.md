# Emotional Intelligence (EQ) Quiz Game

This project is a simple and interactive *Emotional Intelligence (EQ) Quiz* designed to help users understand how they respond to emotions in everyday situations. Instead of theory-based questions, the quiz focuses on real-life scenarios to make it more practical and relatable.

---

## How to Play

1. Start from the *Home Page*.
2. Choose one difficulty level:
   - Easy
   - Medium
   - Hard
3. Each difficulty level contains *10 EQ-based questions*.
4. Read each question carefully and select the option that best reflects an emotionally intelligent response.
5. Click *Next* to move to the next question.
6. After completing all questions, your *final score and feedback* will be displayed.

---

##  Scoring & Feedback

- Each correct answer gives *10 points*.
- Maximum score per level is *100*.

## Features

##  Important Notes

- The difficulty level is selected from the home page.
- If the quiz page is opened directly, it defaults to *Easy* level.
- There is *no negative marking*.
- The quiz is designed for learning and self-awareness, not judgment.
